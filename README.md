# 加密貨幣交易槓桿計算器

這是一個簡單的加密貨幣交易槓桿計算器，可以幫助交易者計算所需的槓桿倍數。

## 功能

- 計算價格變動幅度
- 計算所需槓桿倍數
- 計算實際損失金額

## 使用方法

1. 輸入開倉價位 (USDT)
2. 輸入止損價位 (USDT)
3. 輸入開倉金 (USDT)
4. 輸入可承受損失金額 (USDT)
5. 點擊"計算所需槓桿"按鈕

## 計算公式

- 價格變動幅度 = |止損價位 - 開倉價位|
- 倉位大小 = (可承受損失金額 × 開倉價位) / 價格變動幅度
- 所需槓桿 = 倉位大小 / 開倉金
- 實際損失金額 = (價格變動幅度 × 倉位大小) / 開倉價位
